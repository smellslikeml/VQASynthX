# Ravine Evaluator for VQASynth Reasoning

This directory contains an experimental script for evaluating the quality of spatial reasoning generated by the `VQASynth` pipeline. The methodology is inspired by the rubric-based evaluation framework from the [ResearcherBench](https://github.com/GAIR-NLP/ResearcherBench) project.

## Overview

`VQASynth` produces datasets with rich, structured reasoning in `<think>` blocks. To quantitatively measure the quality of this reasoning, this script uses a powerful Language Model (e.g., GPT-4) as a "judge". The judge assesses the reasoning against a predefined, weighted rubric that covers aspects like logical coherence, object identification, and metric plausibility.

This allows for objective comparisons between different versions of the `VQASynth` data generation pipeline, helping to guide improvements.

## Usage

### 1. Prerequisites

- You have an OpenAI API key with access to a powerful model like `gpt-4-turbo`.
- You have generated a dataset using the main `VQASynth` pipeline. The output should be a JSON file where each item has a `conversations` list.

### 2. Environment Setup

It is recommended to use the same environment as the parent `VQASynth` project. If running standalone, install the necessary dependencies:

```bash
pip install openai
```

Set your API key as an environment variable (recommended):

```bash
export OPENAI_API_KEY="your-openai-api-key"
```

### 3. Run the Evaluation

Execute the script, pointing it to your generated dataset file. You can optionally pass the API key directly.

```bash
python process_evaluation.py --input_file /path/to/your/generated_data.json
```

To evaluate more samples, use the `--limit` flag:

```bash
python process_evaluation.py --input_file ../../data/output/my_dataset.json --limit 50
```

### 4. Check the Output

The script will print the average quality score to the console and save a detailed breakdown of the results in `evaluation_results.json` in the current directory.

```json
[
  {
    "id": "image_001",
    "question": "How far is the red chair from the blue sofa?",
    "reasoning": "Okay, I see a red chair on the left and a blue sofa on the right...",
    "evaluation": {
      "weighted_score": 2.55,
      "details": {
        "Object Identification": 3,
        "Spatial Logic": 3,
        "Metric Plausibility": 2,
        "Groundedness": 2
      }
    }
  }
]
```
