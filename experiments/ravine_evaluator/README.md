# Ravine Evaluator

This script provides a quantitative evaluation of the depth maps generated by the `VQASynth` pipeline. It is inspired by the evaluation methodologies for 3D generative models found in the `tenvoo` repository, adapting concepts like Multi-Scale Structural Similarity (MS-SSIM) to assess the quality of our 3D reconstructions.

## Usage

This evaluator compares a directory of generated depth maps against a directory of corresponding ground-truth depth maps.

1.  **Generate Data:**
    Run the `VQASynth` pipeline on a dataset that has ground-truth depth information (e.g., NYU Depth V2). Ensure the generated depth maps are saved to a specific output directory.

    ```bash
    # Example: Assume VQASynth is configured to save depth maps
    bash run.sh 
    ```

2.  **Run Evaluation:**
    Execute the `process_evaluation.py` script, providing the paths to your generated depth maps and the ground-truth data.

    ```bash
    python experiments/ravine_evaluator/process_evaluation.py \
      --generated_path /path/to/your/generated_depth_maps \
      --ground_truth_path /path/to/your/ground_truth_depth_maps
    ```

## Metrics

The script currently computes and reports:

*   **Average MS-SSIM:** A measure of structural similarity between the generated and ground-truth depth maps. Higher is better (max 1.0).
*   **Average L1 Error:** The mean absolute difference between pixel values. Lower is better.

This tool allows for direct, quantitative comparison between different depth estimation models (e.g., `DepthPro` vs. `VGGT`) within the `VQASynth` ecosystem.
