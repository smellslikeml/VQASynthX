# Experiment: Audio Feature Extraction for Spatial VQA

## Goal
To demonstrate the feasibility of incorporating audio data into the VQASynth pipeline using the MATPAC self-supervised audio model.

## Hypothesis
Audio provides crucial spatial cues (e.g., directionality, distance, environmental context) that can enrich the spatial VQA data generated by VQASynth, helping to resolve visual ambiguities.

## Method
This script uses the pre-trained MATPAC model to extract high-level feature embeddings from a sample audio file. These embeddings could later be fused with visual features from one of VQASynth's stages to generate more complex, multimodal VQA pairs.

## How to Run

1. Navigate to this directory:
   ```bash
   cd experiments/audio_spatial_vqa
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the script:
   ```bash
   python run.py
   ```

The script will download the model checkpoint, generate a test audio file, and print the resulting embedding shapes.
